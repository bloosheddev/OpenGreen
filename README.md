# OpenGreen

This app was created to help people preserve the environment. OpenGreen focuses on recycling and reuse. OpenGreen helps users manage waste wisely and reduce the amount of waste ending up in landfills. OpenGreen features include scans and maps, including a scan to learn how to properly manage waste, and a map to find directions to temporary storage sites. OpenGreen makes it easier for users to make more environmentally friendly decisions when disposing of items.
